Projet Découpeur XML

Contexte

Un collègue de travail reçoit des fichiers XML de recettes ou de dépenses du département, il doit ensuite les rentrer dans un logiciel de comptabilité.

Certains fichiers XML qu'il reçoit comportent beaucoup de balises "PJ" et sont trop volumineux pour le logiciel de comptabilité.

Il me demande de lui créer une application pour découper les fichiers XML.

Les consignes

* Je dois découper le fichier XML en plusieurs parties pour réduire son volume.

* Je dois garder une partie "En Tête" En gardant des balises spécifiques.

* Ensuite je dois le découper en autant de parties que de balises "PJ".

Aperçu de l'interface de l'application

Sur cette interface j'ai mis en place des champs de saisie qui permettent à l'utilisateur de choisir où seront stockés les fichiers découpés.
Une fois le fichier à découper selectionné et le répertoire de destination des fichiers découpés choisis, il suffit de cliquer sur le boutton "Découper".

Aperçu du code première partie

Dans cette première partie du code j'ai utilisé une expression régulière pour définir si le dossier est une dépense ou un recette.
En fonction du résultat de l'expression régulière je passe dans une condition "Si NON". Cela me permet de gérer les "Recettes ou les Dépenses"

Aperçu du code deuxième partie

J'applique mon algorithme à l'aide de boucles pour effectuer mon découpage selon les consignes données.

Aperçu du code troisième partie

Une fois mon algorithme fini et mon fichier découpé, je le sauvegarde dans le dossier prédéfini.
A la fin de cette partie du code on retrouve la fin de la condition "Si NON" du résultat de mon expression régulière.
La suite du code est identique, seul le résultat diffère de la condition de mon expression régulière.
