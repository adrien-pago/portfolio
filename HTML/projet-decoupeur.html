Projet Découpeur XML

Context

Un collègue de travail reçoit des fichier XML de recette ou de dépense du département, il doit ensuite les rentrer dans un logiciel de comptabilité.

Certains fichiers XML qu'il reçoit comporte beaucoup de balise "PJ" et sont trop volumineux pour le logiciel de comptabilité.

Il me demande donc de lui créer une application pour découper les fichiers XML.

Les consignes

* Je dois découper le fichier XML en plusieurs parties pour réduire son volume.

* Je dois garder une partie "En Tête" En gardant des balises spécifiques.

* Ensuite je dois le découper en autant de partie que de balise "PJ".

Apperçu de l'interface de l'application

Sur cette interface j'ai mis en place des champs de saisis qui permettent à l'utilisateur de choisir où sera stockés les fichiers découpés.
Une fois le fichier à découper selectionné et le répertoire de destination des fichiers découpés choisis, il suffit de cliquer sur le boutton "Découper".

Apperçu du code première partie

Dans cette première partie du code j'ai utilisé une expression régulière pour définir si le dossier est une dépense ou un recette.
En fonction du résultat de l'expréssion régulière je passe dans une condition "Si NON". Ce qui me permet de gérer les "Recettes ou les Dépenses"

Apperçu du code deuxième partie

J'applique mon algorithme à l'aide de boucle pour effectuer mon découpage selon les consignes données.

Apperçu du code troisième partie

Une fois mon algorithme fini et mon fichier découpé, je le sauvegarde dans le dossier prédéfini.
A la fin de cette partie du code on retrouve la fin de la condition "Si NON" du resultat de mon expression régulière.
La suite du code est identique juste différère le resultat de la condition de mon expression régulière
